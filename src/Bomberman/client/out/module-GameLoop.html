Module: GameLoop

Methods

(inner) loop(timestamp)

Loop is a constant loop that continously runs while the game is played an updates the visuals on the screen. It views all cells on the board and draws any existing walls, along with rendering players, bombs, and explosions in their current position. Also removes dead entities from the board.

timestamp timestamp Current timestamp.
